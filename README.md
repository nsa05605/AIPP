## 1. CAtNIPP: Context-Aware Attention-based Network for Informative Path Planning
### 논문 소개
- 2차원 AIPP 연구
- GP 기반으로 환경 모델링 및 불확실성 감소를 보상으로 제공
- Attention과 LSTM 구조를 사용하여 그래프 기반 path planning 구조 사용
### 코드 리뷰 진행 상황
- 코드 각 부분에 대한 주석 작성 완료
- 코드 흐름 정리 예정
